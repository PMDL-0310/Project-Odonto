# Hollywood Odonto

## O que foi feito

Website profissional para clínica odontológica com:

- Página inicial responsiva com informações sobre a clínica
- Seção de serviços oferecidos (limpeza, clareamento, implantes, ortodontia, etc)
- Formulário interativo para agendamento de consultas
- API REST com endpoints para dados de serviços, informações da clínica e agendamentos
- Design moderno e responsivo para todos os dispositivos
- Navbar com menu mobile
- Integração completa entre frontend e backend com Node.js/Express
